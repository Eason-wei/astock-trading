## 5.1 题材炒作的本质

### 核心定义
```
题材 = 有想象空间的板块/概念
     = 资金愿意聚集的理由
     = 赚钱效应的载体

题材的三个维度：
1. 新不新（第一次出现 vs 老题材反复炒）
2. 大不大（容量能否容纳大资金）
3. 强不强（有没有龙头带队，有无跟风）
```

### 养家心法核心
```
"交易的本质是群体博弈"
"把握市场情绪，才能把握利润"
"高手买入龙头，超级高手卖出龙头"
```

### 题材炒作周期
```
酝酿期 → 启动期 → 蔓延期 → 高潮期 → 退潮期
  ↓         ↓         ↓         ↓         ↓
资金建仓   龙头涨停   跟风扩散   全面爆发   资金出逃
```

## 5.2 资金流向逻辑

### 资金流动的本质
```
trigger: 有赚钱效应
mechanism: 钱会往能赚钱的地方流
outcome: 资金聚集 → 股价上涨 → 赚钱效应扩散

trigger: 有亏钱效应
mechanism: 钱会逃离亏损的地方
outcome: 资金撤离 → 股价下跌 → 亏钱效应扩散
```

### 主力资金信号（when_works）
```
trigger → 判断：
放量+上涨 → 主力在买入（真实上涨）
缩量+上涨 → 锁仓（可能持续）
放量+下跌 → 主力在出货（警惕）
缩量+下跌 → 正常（还没跌完）
```

### 北向资金观察
```
北向净流入 → 外资看好A股
北向净流出 → 外资不看好A股
+ 外资流向与大盘正相关
+ 外资偏好：蓝筹/消费/科技龙头
```

## 5.3 板块轮动规律

### 四阶段轮动
```
启动（冰点后）：
  → 资金试探新题材
  → 龙头率先涨停
  → 特征：板块内只有1-2只涨停

蔓延（确认后）：
  → 跟风股批量涨停
  → 板块效应形成
  → 特征：板块内3只以上涨停

高潮（加速后）：
  → 板块全面爆发
  → 低位股补涨
  → 特征：板块成交额放大

退潮（龙头断板后）：
  → 资金开始撤出
  → 切换新方向
  → 特征：龙头跌停 or 缩量滞涨
```

### 题材容量判断
```
容量大（百亿+）：能容纳大资金，适合机构参与
容量小：只能容纳游资，容易一字板买不到

容量判断标准：
  + 有千亿市值的中军股涨停 → 容量大
  + 全是小市值涨停 → 容量小，游资行为
```

### 题材共振判断
```
共振 = 板块 + 指数 + 情绪 同时向上
共振强 = 成功率更高
无共振 = 独立行情，持续性存疑
```

## 5.4 量价关系

### 放量 vs 缩量
```
放量上涨：
  → 有资金主动买入，看涨
  → 如果在高位放量滞涨 = 主力出货

缩量上涨：
  → 锁仓行为，抛压小
  → 上涨可持续
  → 但缩量到极点可能反转

放量下跌：
  → 有资金出逃，看跌
  → 如果在低位放量 = 主力在吸筹

缩量下跌：
  → 还没跌完，卖盘未出清
  → 但缩量到地量可能见底
```

### 换手率信号
```
换手率突然放大 → 资金异动，关注
换手率持续高位 → 趋势延续
换手率萎缩 → 趋势衰竭

次新股/高位股高换手 → 警惕
低位股温和换手 → 健康
```

### 量比选股
```
量比>1.5 + 涨幅3-5% = 异动信号
量比>3 = 巨量异动，谨慎
量比<0.5 = 缩量，方向不明
```

## 5.5 涨停板结构

### 涨停板分类
```
硬板（一字板）：
  → 一笔封死，无换手
  → 次日溢价高，但买不到
  → 风险：开板即大面

换手板（实体涨停）：
  → 充分换手
  → 次日溢价相对稳定
  → 风险：烂板容易炸

尾盘板：
  → 14:00后涨停
  → 主力实力存疑
  → 可能是套利，谨慎

地天板：
  → 从跌停拉到涨停
  → 撬板资金介入
  → 高风险高回报
```

### 封单强度
```
封单金额/流通市值 > 1% → 强
封单逐渐变小 → 弱，可能开板
封单突然撤单 → 主力诱多
```

## 5.6 游资思维

### 游资操作逻辑
```
养家心法：
"买入机会，卖出风险"
"只做对的交易，胜负交给概率"

核心：
1. 预判 → 试错 → 确认 → 加仓
2. 第一时间发现机会
3. 错了立即止损，不抱幻想
4. 赢钱时加仓，输钱时减仓
```

### 弱势反弹策略（养家）
```
阶段1（初期）：
  → 做强势股回调反抽
  → 或新题材首板

阶段2（中期）：
  → 做超跌股，在连续下跌后再度猛跌时买

阶段3（末期）：
  → 做符合主流热点的强势股
  → 市场情绪开始转暖
```

### 养家心法核心理念
```
1. 本人理论体系的核心思想：
   基于对市场情绪的揣摩
   判断风险和收益的比较
   指导实际操作

2. 持币者与持筹者的博弈：
   股价上涨 → 持筹者倾向卖出
   持币者不敢买 → 压力
   股价下跌 → 持币者愿意买
   持筹者不愿卖 → 支撑

3. 群体博弈：
   知道自己在群体中的位置
   知道群体的行为模式
   才能在博弈中占优
```

## 5.7 消息催化逻辑

### 消息分类
```
政策消息：级别最高，持续性最强
          （降准/产业政策/领导人讲话）

业绩消息：确定性最高
          （业绩超预期/公告利好）

产业消息：中期影响
          （产品价格变化/技术突破）

市场传闻：不确定，警惕
          （小道消息/截图传播）
```

### 消息影响路径
```
消息出现
  → 资金分析影响程度
  → 资金做出买入/卖出决策
  → 股价开始变动
  → 群体跟进
  → 加速/反转

时滞：
  政策消息 → 下一个交易日反应（可能高开低走）
  盘中消息 → 当日即时反应
```

### 消息催化时机
```
买预期：
  → 在消息正式公布前买入
  → "靴子落地"前卖出

卖事实：
  → 消息公布后反而卖出
  → 因为"利好出尽是利空"
```
