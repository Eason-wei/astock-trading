> 核心问题：如何不亏大钱，如何在错误时快速止损。

> 核心参考：框架v1.3 Step 9.5系统风控 + 止损铁律

## 4.1 止损是呼吸

### 核心原则
```
止损不是运气坏，是交易的成本。
不止损一次，够你止损十次。
```

### 止损标准
```
单票止损：-3% 触发止损（无条件执行）
连板止损：断板即清仓（不幻想反弹）
系统止损：Step 9.5任一项触发 → 仓位≤10%
```

### 止损vs不止损的后果
```
不止损：亏10% → 需要赚11%回本
不止损：亏30% → 需要赚43%回本
不止损：亏50% → 需要赚100%回本

止损3次各亏3% → 需要赚9.3%回本（可控）
不止损1次亏30% → 需要43%（很难）
```

## 4.2 核按钮认知

### 什么是核按钮
```
trigger: 当日买错/打板失败，次日集合竞价直接挂跌停价卖出
mechanism: 
  - 资金量大、态度坚决、时间短促
  - 同质化恐慌引发群体效应
  - 将股价压在跌停板上

outcome:
  - 成功：止损果断，不影响心态
  - 失败：引发板块恐慌，破坏短线生态
```

### 核按钮的目的
```
trigger: 买入后走势不符预期
mechanism:
  1. 次日集合竞价卖出 → 不妨碍追新标的（仓位可倒腾）
  2. 避免亏钱心态影响后续操作

outcome:
  核按钮不是"损人不利己"
  是用小亏损避免大亏损的止损策略
```

### 核按钮何时触发
```
该核时核：
  - 买入后当日炸板，次日低开
  - 明显判断错误，逻辑破坏
  - 大盘系统性风险

不该核时不核：
  - 龙头首阴（可能是分歧，不是破坏）
  - 缩量回调（可能是洗盘）
  - 跟风股小亏（可能是轮动）
```

## 4.3 系统风控（Step 9.5）

| 检查项 | 阈值 | 触发后 |
|--------|------|-------|
| 市场宽度 | 上涨家数<25% | 仓位≤10% |
| 流动性 | 成交额<8000亿 | 仓位≤10% |
| 恐慌情绪 | 跌停>30家 | 仓位≤10% |
| 趋势破坏 | 指数破60日线 | 仓位≤10% |
| 宏观事件 | 重大利空 | 仓位≤10% |
| 异常情况 | 千股涨跌停 | 仓位≤10% |

**≥1项触发 → 立即执行，全部≤10%**

## 4.4 心态纪律

### 三大铁律
```
1. 空仓是高级操作
   → 当快速诊断模糊或系统风险触发时，主动空仓是成功

2. 止损是呼吸
   → 止损价必须无条件执行，纪律是连接框架与盈利的唯一桥梁

3. 慢即是快
   → 放弃无效交易，只在胜率高、盈亏比优时出手
```

### 恐惧与贪婪的控制

**恐惧的成本：**
```
- 回避好的交易机会
- 在高波动时犹豫不决
- 过早退出盈利仓位
```

**贪婪的成本：**
```
- 持有到亏损（"再等等就回来了"）
- 重仓在高位
- 不止损，幻想反弹
```

### 知行合一
```
知道 ≠ 做到
做到 ≠ 持续做到

知行合一的标准：
不是在"简单时"做到，而是在"困难时"也能做到。
```

### 复盘自问（框架Step 12）
```
1. 我是否因"龙头信仰"想低吸？（确认偏误）
2. 我是否因持仓而过度看好？（持仓偏见）
3. 我是否因连续踏空而焦虑想追高？（FOMO）
4. 我是否因亏损而沮丧想报复交易？（情绪化）
5. 我是否因盈利而兴奋想加大仓位？（过度自信）
```

## 4.5 决策疲劳管理

### 决策疲劳指数
```
累计决策数量 × 决策复杂度：
- 0-10个：正常
- 10-20个：轻微疲劳，警惕
- 20-30个：中度疲劳，停止开新仓
- >30个：高度疲劳，强制空仓一天
```

### 决策疲劳来源
```
1. 频繁开关仓
2. 同时关注太多股票
3. 非计划内的临时决策
4. 亏损后强行交易
```

---

## 4.6 执行偏差归因

当交易计划执行偏差时，深挖根源：

| 偏差类型 | 根源 | 应对 |
|---------|------|------|
| 恐惧不止损 | 过度规避损失心理 | 降低单票仓位 |
| 贪婪不止盈 | 过度自信/侥幸心理 | 设置机械止盈 |
| 焦虑追高 | FOMO（错失恐惧症） | 接受踏空 |
| 懒惰不盯盘 | 缺乏纪律 | 提前挂单 |
| 怀疑系统 | 系统信任危机 | 暂停交易，找回节奏 |

## 4.7 风险管理决策树

```
买入前：
├─ 风险收益比≥3:1？ → 否 → 不开仓
├─ 期望值>1%？ → 否 → 不开仓
└─ Step 9.5有触发？ → 是 → 仓位≤10%

买入后：
├─ 跌破止损价 → 立即止损，不幻想
├─ 触发核按钮条件 → 次日集合竞价出
└─ 大盘系统性风险 → 立即降仓

持仓中：
├─ 连续3日亏损 → 暂停交易，复盘反思
├─ 精力不足/情绪波动 → 降低仓位
└─ 账户回撤>10% → 暂停交易
```
