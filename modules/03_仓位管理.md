## 3.1 情绪周期仓位表（基础仓位）

| 周期 | 涨停家数 | 最高板 | 跌停家数 | 基础仓位 | 操作 |
|------|---------|--------|---------|---------|------|
| **启动** | 50-70 | 3-4板 | <10 | 20-30% | 轻仓试错主线 |
| **发酵** | 70-100 | 4-5板 | <10 | 50-70% | 重仓聚焦主线 |
| **高潮** | >100 | 5板+ | <15 | 30-50% | 持有，不开新仓 |
| **分歧** | 50-70 | 断板 | 15-30 | 10-30% | 仅低吸绝对龙头 |
| **退潮** | <50 | ≤3板 | >15 | 0-10% | 空仓/极小仓 |

> **框架v1.3 Step 9.5触发时，全部作废，强制≤10%**

## 3.2 仓位调整系数

### 系数1：账户净值
| 条件 | 系数 | 说明 |
|------|------|------|
| 新高附近 | 1.0 | 正常操作 |
| 回撤5%以内 | 0.8 | 适度防守 |
| 回撤5-10% | 0.5 | 大幅减仓，保命为主 |
| 回撤>10% | 0.2 | 暂停操作，反思调整 |

### 系数2：市场风险偏好
| 条件 | 系数 |
|------|------|
| 高风险偏好 | 1.0-1.2 |
| 低风险偏好 | 0.7-0.9 |

### 系数3：个人状态
| 条件 | 系数 |
|------|------|
| 精力充沛，执行力强 | 1.0 |
| 疲惫，情绪波动大 | 0.5-0.8 |

### 最终仓位公式
```
最终仓位 = 情绪周期基础仓位 × 系数1 × 系数2 × 系数3
（所有系数≤1，最终仓位不超过基础仓位上限）
```

## 3.4 竞价仓位策略

### 竞价看什么
```
关键信号（trigger → action）：
1. 竞价从涨停杀到水下 → 主力在出货，不买
2. 竞价从水下一路拉升至水上 → 抢筹信号，可买
3. 竞价超预期爆量高开 → 当日强势，追加
4. 昨日强势股竞价大幅低开 → 规避，等修复
5. 竞价末端爆量 → 当日方向明确
```

### 竞价陷阱
```
9:15-9:20可以撤单 → 主力可能在诱多
9:19最后几秒撤单 → 虚假涨停信号
尾盘偷袭板次日竞价差 → 不接力
```

### 竞价买入条件
```
1. 昨日涨停股，今日竞价高开（>3%）且未跌破水位
2. 有板块效应（跟风股也在拉升）
3. 大盘开盘不差（不是系统性风险日）
```

## 3.5 分批建仓策略

### 标准分批
```
首仓：20%（确认方向）
二仓：30%（确认持续性）
三仓：50%（行情确认后重仓）
```

### 风险收益比仓位
```
风险收益比≥3:1 → 正常仓位
风险收益比2-3:1 → 减半仓
风险收益比<2:1 → 不开仓
```

---

## 3.6 仓位管理核心原则

```
1. 好机会重仓，没有好机会轻仓或空仓
2. 不拘泥于手中持仓，要看全局板块走势
3. 不应只守着弱势股，应向强势股集中
4. 大盘下跌时，用轻仓练盘感，不是为了赚钱
5. 补仓只在两种情况下：大盘稳 + 个股强
```

### 养家心法仓位逻辑
```
有赢面的股票 → 重点做
  - 主流：反复做，高抛低吸
  - 次主流：绝佳机会才重仓
  - 非主流：轻仓，隔日超短

没有好机会 → 耐心等待 or 轻仓练盘感
```
