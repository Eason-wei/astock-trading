## 6.1 每日复盘流程（Step 1 → Step 6）

### Step 1：T-1数据（昨日涨停）
```
T-1涨停板块：
| 题材 | 涨停数 | 代表股 | 最高板 | 梯队 | 5日量比 |
T-1最高板：
T-1断板股：（用于亏钱效应监测）

T-1关键数据：
  - 涨停总数
  - 连板总数
  - 跌停总数
  - 炸板率
  - 昨日涨停今溢
```

### Step 1.5：三问定乾坤（5分钟）
```
1. 空间板生死：龙头晋级/断板/跌停？
2. 主线状态：加强/分歧/退潮？
3. 亏钱效应：修复/核按钮/扩散？
```

### Step 2：T日开盘情绪
```
- 昨日涨停今溢：>5%积极，<0%消极
- 昨日断板反馈：核按钮/修复/横盘
- 开盘涨跌比
```

### Step 3：T-1→T题材延续性
```
梯队健康标准：≥3首板→≥2二板→≥1三板
成交额：>1.2倍5日均量=放量，<0.8倍=缩量
晋级率：2→3板成功率>50%=健康
```

### Step 4：T日新题材
```
新题材判断标准：
  + 有容量：百亿级中军股涨停
  + 有共振：与指数同步放量启动
  → 否则：一日游，不碰
```

### Step 5：合并同类项
```
同题材T-1 + T合并
新题材单独标记
```

### Step 6：全题材总览表
```
| 题材 | 来源 | T-1 | T日 | 变化 | 成交额 | 梯队 | 共振 | 结论 |
```

## 6.2 连板分析+亏钱效应（Step 7）

### 连板晋级率
```
量化：
  2→3板成功率 >50% = 接力健康
  2→3板成功率 <30% = 接力枯竭

连板晋级率 = 成功晋级的连板股数 / 昨日2板以上总数
```

### 亏钱效应监测
```
关注：
  - 昨日断板股今日反馈
  - 涨停股次日平均最大回撤
  - A杀个股（连续跌停）
```

### 最高板信号
```
最高板龙头断板：
  + 跟风股也跌 → 退潮
  + 跟风股不跌 → 内部轮动（切换到跟风）

最高板涨停：
  + 梯队健全 → 主升
  + 梯队断层 → 分歧
```

## 6.3 市场结构与风格（Step 9）

### 市场风格判断
```
风格类型：
  - 连板接力：连板股平均高度≥4板，>30%涨停股是连板
  - 趋势波段：趋势股（20日涨>30%）数量多，平均换手<10%
  - 混合：轮动快，无明显风格

操作匹配：
  连板接力 → 打板、接力
  趋势波段 → 低吸趋势龙头
  混合 → 轻仓短线
```

### 黄白线判断
```
黄线（中小盘）在白线（权重）上方：
  → 中小盘活跃，赚钱效应好
  → 重个股轻大盘

白线在黄线上方：
  → 权重护盘，个股弱
  → 谨慎操作
```

## 6.5 盘中三步检查（Step 11）

### 一查情绪锚点
```
1. 空间板竞价：晋级/断板/跌停？
2. 昨日断板股反馈：是否核按钮？
3. 结论：与预期一致 → 继续；不符 → 暂停
```

### 二查主线验证
```
1. 计划主线开盘强度：加强 or 弱化？
2. 计划新题材开盘分化程度：大幅分化？
3. 结论：符合计划 → 执行；不符 → 等待
```

### 三查风控触发
```
1. Step 9.5各项：是否瞬间恶化？
2. 跌停家数是否突然增加？
3. 结论：触发 → 立即降仓，停止开新仓
```

**三者皆符 → 执行计划**
**任一不符 → 暂停，重新评估**

## 6.6 突发应对

| 突发情况 | 应对 |
|---------|------|
| 突发利空 | 持仓股立即止损，目标股取消 |
| 指数突然跳水 | 停止开新仓，持仓跌破止损立即出 |
| 板块突然拉升（非主线） | 不追高，等次日确认 |
| 个股突然直线拉升（非计划） | 不追高，放弃 |

## 6.7 盘感训练

### 每日训练方法
```
1. 坚持复盘：每天花2小时复盘
2. 选出异动股：按量比/换手率/涨幅筛选
3. 重点浏览：目标个股的详细信息
4. 记录判断：写下自己的预判
5. 次日验证：对比预判与实际
```

### 盘感来源
```
来自：每天坚持复盘
     + 对异动股的持续跟踪
     + 对预判的验证和修正
     + 理解市场背后的资金逻辑
```

## 6.8 预判与跟随

### 预判 → 试错 → 确认 → 加仓（养家心法）
```
预判：基于分析预判方向
试错：用轻仓试探
确认：方向正确后加仓
加仓：确认后可重仓
```

### 跟随 vs 预判
```
跟随：
  → 确认后再买
  → 确定性高，但成本也高

预判：
  → 在信号出现前预判
  → 成本低，但可能错
```

### 跟随策略
```
只做绝对龙头：
  → 无龙不操作
  → 有龙就跟随

龙头确认信号：
  → 连续涨停（2板以上）
  → 带动跟风股涨停
  → 板块内有资金共识
```
