# 模块一：择时体系

> 核心问题：什么时候可以出手，什么时候必须空仓。
> 整理自：learnings.json 中 情绪周期(22) + 节点把握(20) + 弱势市(23) + 外围影响(29) 条目
> 筛选原则：只保留有实质因果逻辑的条目，废话/新闻/股评已过滤

## 1.1 情绪周期五阶段

### 阶段定义与量化标准

| 阶段 | 涨停家数 | 最高板 | 连板率 | 跌停家数 | 炸板率 | 典型特征 |
|------|---------|--------|-------|---------|-------|---------|
| **启动** | 50-70 | 3-4板 | >50% | <10 | <25% | 连板开始出现，空间打开 |
| **发酵** | 70-100 | 4-5板 | >50% | <10 | <25% | 梯队健全，主线明确 |
| **高潮** | >100 | 5板+ | >60% | <15 | <20% | 全面爆发，板块普涨 |
| **分歧** | 50-70 | 断板 | 30-50% | 15-30 | >35% | 龙头断板，亏钱效应出现 |
| **退潮** | <50 | ≤3板 | <30% | >15 | >40% | A杀个股，核按钮频现 |

**trigger（如何判断当前阶段）：**
- 启动：昨日连板股今日晋级率>50%，无核按钮
- 发酵：主线板块成交额放大，梯队完整（3→2→1板健全）
- 高潮：涨停家数创近期新高，市场普涨
- 分歧：最高板龙头断板，跌停股开始增多
- 退潮：连续2日跌停>15家，空间板≤3板

**mechanism（为什么这个阶段会这样）：**
- 启动期：空仓资金看到赚钱效应，开始试探龙头股
- 发酵期：赚钱效应扩散，跟风股批量拉升
- 高潮期：场外资金看到巨大赚钱效应，加速入场
- 分歧期：先手资金开始兑现，筹码松动
- 退潮期：亏钱效应扩散，持股者恐慌，抄底者被埋

**仓位对应（框架v1.3 Step10）：**
- 启动：20-30%
- 发酵：50-70%
- 高潮：30-50%（持有不追）
- 分歧：10-30%（仅低吸绝对龙头）
- 退潮：0-10%

### 关键判断信号

#### 冰点反转信号（退潮→启动）
```
trigger: 连续3日涨停<40家，跌停>20家，空间板≤3板
outcome: 新题材出现且容量足够（百亿中军涨停）+ 共振指数
when_works: 情绪长期低迷后的首次逆转，胜率最高
when_fails: 假反转，冰点后还有冰点（无量反弹）
```

#### 高潮次日分化信号（高潮→分歧）
```
trigger: 前一日涨停>100家，连板率>60%
outcome: 今日涨停缩减、跌停增加、龙头断板
when_works: 缩量加速板次日必分
when_fails: 超级主线（政策强催化）可延续2-3日
```

#### 龙头断板信号（主升→分歧）
```
trigger: 最高板龙头断板（非跌停，是放量换手）
outcome: 板块可能退潮，但也可能内部轮动（资金切换到跟风股）
when_works: 跟风股也跟随下跌 = 退潮
when_fails: 跟风股不跌，龙头断板后次日反包 = 内部轮动
```

---

### 龙空龙战法（择时核心）

**核心理念：**
```
只做绝对龙头，无龙则空仓
```

**触发条件（trigger）：**
1. 存在市场总龙头（2-3板以上确认）
2. 情绪周期处于启动或发酵期
3. 主流板块明确，梯队健全

**退出条件（trigger）：**
1. 龙头断板 → 情绪退潮信号
2. 跌停家数>20家
3. 计划主线出现大分化

**mechanism：**
- 情绪上升期：资金追逐龙头，龙头溢价最高
- 情绪退潮期：龙头补跌，跟风股率先崩塌
- 龙空龙本质：在赚钱效应最强时重仓，在亏钱效应出现时空仓

## 1.2 节点把握

### 买入节点

#### 弱转强节点
```
trigger: 个股/板块经历分歧后，次日竞价高开或开盘快速拉升
mechanism: 分歧日被压制，次日资金主动做多，说明多方更强
outcome: 成功则涨停，失败则回到分歧价位
when_works: 板块梯队完整，龙头地位已确立
when_fails: 独立走势，无板块共振
```

#### 尾盘买入节点
```
trigger: 临收盘前30-60分钟，股价稳定在日内高位
mechanism: 避开当日不可卖出风险，次日开盘可自主决策
outcome: T+1开盘即知涨跌
when_works: 强势股回调后尾盘拉升
when_fails: 弱势市场，尾盘可能出现偷袭跳水
```

#### 分歧转一致节点
```
trigger: 开盘分歧（高开低走或低开），午后资金回流拉升
mechanism: 早盘分歧清洗浮筹，午后拉升说明抛压被消化
outcome: 隔日高开或连板
when_works: 成交额未明显放大（抛压轻），情绪不差
when_fails: 大盘午后跳水，分歧转一致的逻辑消失
```

---

### 卖出节点

#### 分时放量滞涨
```
trigger: 股价在高位，但成交量突然放大（超过均量2-3倍），价格不创新高
mechanism: 主力在出货，承接资金不足
outcome: 当日可能炸板或次日低开
when_works: 缩量涨停后次日放巨量滞涨
when_fails: 强势连板股，因锁仓筹码少所以放量正常
```

#### 跌破均价线
```
trigger: 分时图股价持续在均价线下方运行
mechanism: 均价线是多空分水岭，跌破代表空方主导
outcome: 当日难以收回，次日可能低开
when_works: 非强势股
when_fails: 洗盘吸筹阶段可能短暂跌破均价线
```

## 1.3 弱势市/大盘择时

### 大盘黄白线判断
```
trigger: 分时图黄线（中小盘）在白线（权重）上方运行
mechanism: 中小盘个股活跃，赚钱效应存在，可以重个股轻大盘
outcome: 市场不缺少机会，操作环境较好
when_works: 反弹行情中黄线在白线上方
when_fails: 二八分化，赚钱效应集中在少数权重股
```

### 大盘与个股关系
```
trigger: 大盘下跌，个股处于弱势
mechanism: 大盘环境差，个股很难独善其身
outcome: 降低操作频率或空仓
when_works: 大盘处于20日均线下方
when_fails: 极端二八行情，有基本面支撑的个股可能逆势
```

### 系统性风险判断（Step 9.5触发项）
| 指标 | 阈值 | 含义 |
|------|------|------|
| 上涨家数比 | <25% | 市场宽度极窄 |
| 成交额 | <8000亿 | 流动性枯竭 |
| 跌停家数 | >30家 | 恐慌蔓延 |
| 指数破60日线 | 是 | 趋势破坏 |

**触发任一项 → 仓位≤10%**

## 1.4 外围影响

### 传导逻辑
```
trigger: 美股/港股大幅波动
mechanism: A股开盘跟随，A股有"跟跌不跟涨"的历史特征
outcome: A股大幅低开，但能否独立取决于内部资金
when_works: 地缘冲突+全球恐慌 → A股难以独善其身
when_fails: A股自身逻辑强（如政策刺激）→ 可能低开高走
```

### 黄金走势
```
trigger: 地缘冲突升级 → 黄金上涨
mechanism: 避险资金流入黄金 → 其他风险资产承压
outcome: A股相应板块（黄金/军工/能源）可能逆势
when_works: 纯粹避险情绪（非经济衰退）
when_fails: 经济衰退预期 → 所有人都在卖
```

### 关键变量
```
1. 北向资金流向（外资情绪指标）
2. 港股走势（与A股联动最紧密）
3. 美股期指（隔夜A股开盘情绪）
4. 汇率（人民币贬值压力）
```

## 1.5 择时体系核心清单

**每日必查（开盘前5分钟）：**
1. 昨日涨停今溢：>5%积极，<0%消极
2. 昨日断板股反馈：核按钮？修复？
3. 空间板：晋级？断板？跌停？
4. 跌停家数：是否>15家
5. 大盘黄白线：谁在领涨

**诊断结果→操作：**
- ✅ 全正 → 正常仓位，按计划执行
- ⚠️ 1-2项异常 → 降仓至50%，减少开新仓
- ❌ ≥3项异常 → 仓位≤10%，停止开新仓
