## 2.1 龙头股识别

### 龙头股的本质
```
龙头股是一段赚钱效应的载体。
龙头涨→板块跟→资金涌入→赚钱效应扩散
龙头死→板块死→亏钱效应扩散
```

### 识别三维（trigger）
| 维度 | 具体标准 | 权重 |
|------|---------|------|
| **板块带动性** | 龙头涨→同板块股票也涨 | 必须 |
| **先于板块启动** | 龙头先涨停，板块后启动 | 重要 |
| **领涨且抗跌** | 涨时领先，跌时抗跌 | 重要 |
| **持续创新高** | 调整相对较小，始终不破位 | 必须 |

### 量化标准
```
trigger:
  - 连板数 ≥ 2板（2板是龙头的门槛确认）
  - 流通市值：10-100亿（太大拉不动，太小容易被操控）
  - 封单金额/流通市值 > 1%（封单强度）
  - 涨停时间：早盘（10:30前）优于尾盘
  - 大盘下跌末端，逆市涨停
```

### 龙头身位判断
```
龙头必须从涨停板开始。不涨停的股票不可能做龙头。
涨停板是多空双方最准确的攻击信号。

二板定龙头：第一板可能是个股行为，第二板才是真正的板块共振。
```

### when_works / when_fails
```
when_works:
  - 有板块效应：龙头涨停能带动3只以上跟风
  - 有消息催化：政策/业绩/产业利好
  - 情绪周期处于启动或发酵期
  - 流通市值适中，方便资金接力

when_fails:
  - 独立涨停，无跟风
  - 无消息支撑，纯筹码博弈
  - 情绪退潮期，龙头补跌
  - 高位（>7板）筹码断层
```

## 2.2 首板战法（一进二）

### 核心逻辑
```
trigger: 个股在三个月内第一次涨停（首板）
mechanism: "动量惯性"+ "资金共识强化"
outcome: 3日内创新高概率>70%，30日>86%，90日>90%
```

**数据来源：量化实验（雪球用户"难为一点"）**

### 一进二筛选标准

| 指标 | 理想标准 | 陷阱 |
|------|---------|------|
| 涨停时间 | 10:30前 | 14:00后尾盘板不碰 |
| 封单强度 | 封单金额/流通市值>1% | 封单越来越小 |
| 封板质量 | 硬板（一笔封死） | 烂板（反复开板） |
| 成交量 | 温和放量（<3倍均量） | 巨量（分歧过大） |
| 跟风股 | 有3只以上跟风涨停 | 独立涨停 |
| 消息面 | 有利好催化 | 无原因涨停 |

### 首板优势（mechanism）
```
1. 成本低：相对连板股，首板位置低，成功连板有成本优势
2. 止损小：位置低，失败回撤-10%以内（vs连板-15%以上）
3. 流动性好：失败时出货容易，连板失败直接流动性枯竭
4. 位置低：获利盘小，下跌不会特别凶狠
```

### 首板劣势
```
1. 目标太多：每天首板几十只，需要瞬间判断优劣
2. 一日游风险：没有板块效应的首板次日容易被埋
3. 需要筛选：只做有板块效应的首板
```

### 两种优质首板类型
```
类型A：指数共振首板
trigger: 指数止跌回升，个股率先涨停
mechanism: 大盘环境转暖，个股借势
outcome: 溢价较高，次日容易高开

类型B：新周期切换首板
trigger: 老龙头见顶，资金高低切换
mechanism: 资金主动试盘新方向
outcome: 可能是新周期龙头
```

## 2.3 连板接力（二板及以后）

### 二板定龙头
```
trigger:
  - 首板后次日高开/快速拉升晋级二板
  - 换手板优先（一字板无换手，无法判断）
  - 成交额放大（≥首板1.5倍）
  - 换手率：15%-25%（筹码健康）
  - 跟风股存在（板块内多只涨停）

mechanism:
  二连板是对首板逻辑和强度的有效确认。
  资金用真金白银投票，证明了这只股票的"动量惯性"。

outcome:
  成功：主升浪开启
  失败：回到首板价位附近（-10%以内）
```

### 连板晋级率
```
量化标准（框架Step7）：
  2→3板成功率 >50% = 接力健康
  2→3板成功率 <30% = 接力枯竭

when_works:
  - 情绪处于启动或发酵期
  - 板块梯队完整（3→2→1板健全）
  - 跟风股能持续

when_fails:
  - 情绪高潮次日（缩量加速板必死）
  - 梯队断层（只有龙头，没有跟风）
  - 缩量一字板（持筹者不愿卖，无法换手）
```

### 接力失败典型形态
```
1. 缩量一字板后炸板（无换手，持筹者一次性砸盘）
2. 高位放量长阴（主力出货）
3. 跟风股率先下跌（板块退潮信号）
```

## 2.4 龙头二波（龙回头）

### 触发条件
```
trigger:
  - 龙头经历一波上涨后，短期回调20-30%
  - 新题材没有起来抢夺资金
  - 龙头在回调中没有破坏均线结构
  - 出现止跌信号（缩量+长下影线）

mechanism:
  主力资金在大幅拉升后，短线获利丰厚。
  股价下跌后吸引抄底资金，形成二波预期。

outcome:
  成功：突破前高，开启第二波
  失败：均线压制，反弹结束
```

### 二板首阴（龙头分歧买点）
```
trigger:
  - 二连板后出现第一根阴线
  - 假阴线优先（收盘涨但实际跌）
  - 跌幅3-7%
  - 不跌停，不破5日线

mechanism:
  二板定真龙后，分歧洗盘。
  多方在分歧中守住阵地，次日反包。

when_works:
  - 跟风股没有跌（证明板块没退潮）
  - 成交额温和放量（<3倍）
  - 有板块效应

when_fails:
  - 跟风股大跌（板块退潮）
  - 跌停（空方力量过强）
  - 巨量（主力在出货）
```

## 2.5 半路战法

### 适用场景
```
trigger: 强势股回调后，在拉升过程中买入（非涨停价买）
mechanism: 在"确定性的基础上"追求更大空间
outcome: 成功则当日有浮盈，失败则当日被套

when_works:
  - 盘感极强，能在分时图识别启动点
  - 板块正在拉升，个股跟风
  - 成交额配合放大

when_fails:
  - 盘感不够，无法判断是反弹还是反转
  - 个股独立走势，无板块共振
  - 弱势市场，半路板容易被砸
```

### 核心要求
```
1. 必须是已经涨停过的强势股（有过"认证"）
2. 必须在拉升过程中买，不是低位埋伏
3. 盘感要求极高，新手不建议做
4. 止盈止损都要在当日完成
```

## 2.6 选股因子体系

### 核心因子（量价）

| 因子 | 短线参考值 | 含义 |
|------|-----------|------|
| 换手率 | >3% | 活跃度高，流动性好 |
| 量比 | >1.5倍 | 放量异动，关注 |
| 涨幅空间 | 3-5% | 上涨空间适中 |
| 流通市值 | 50-100亿 | 方便资金进出 |
| 净利润 | >0 | 基本面要求 |

### 换手率与行情关系
```
换手率突然放大 → 资金异动，可能启动
换手率持续高位 → 趋势延续
换手率萎缩 → 趋势衰竭
缩量涨停 → 锁仓，次日可能继续
放量涨停 → 换手接力，成功率更高
```

### 量比选股
```
量比>1.5 + 涨幅3-5% + 换手>3% + 流通50-100亿
= 短线异动股，需要进一步判断方向
```

## 2.7 选股体系决策树

```
第一步：判断情绪周期（择时）
├─ 退潮期 → 空仓，不选股
├─ 分歧期 → 只做龙头低吸
├─ 启动/发酵 → 可做一进二、首板
└─ 高潮期 → 持有不追，降低仓位

第二步：判断板块方向（题材）
├─ 主线明确 → 聚焦主线，放弃杂毛
└─ 无主线 → 空仓或极小仓试错

第三步：选个股
├─ 龙头已确立 → 买龙头，不买跟风
├─ 龙头未确立 → 买一进二，等二板确认
└─ 首板筛选 → 时间早/封单强/有跟风/有催化

第四步：检查when_fails
├─ 无板块共振 → 不碰
├─ 高位缩量一字 → 不碰
└─ 跟风股已掉队 → 放弃
```
