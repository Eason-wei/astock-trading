## §4 · 亏钱效应分析 (pain_effect_analyzer)

### 源码关键逻辑 (pain_effect_analyzer.py)

```python
# 核心否决条件 (已收紧版本):
def _check_veto(self, fupan: Dict, lianban: Dict) -> Tuple[bool, str, List[str]]:
    zhaban = len(lianban.get('zhaban', []))
    top_rate = fupan['top_rate']
    long_ban = fupan['long_ban']

    # ① 极端否决: 炸板率高 + 连板高 = 高位炸板
    if zhaban >= 15 and top_rate <= 70 and long_ban >= 6:
        return True, 'veto_1', ['极端否决①']

    # ①b 高温警示: 炸板率高 + 连板中高 = 降权10分
    if zhaban >= 10 and top_rate <= 75 and long_ban >= 5 and self.degree >= 55:
        self.score = max(0, self.score - 10)
        veto_reasons.append('veto_1b_warn')
```
