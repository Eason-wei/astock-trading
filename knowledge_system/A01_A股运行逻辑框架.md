# A股运行逻辑框架 v1.0

> 目标：理解「钱为什么来、怎么流、流多久」，而不是记忆招式。

---

## 一、总纲：钱是流向最小阻力的地方

A股的本质是**政策市 + 资金市 + 情绪市**的三位一体。

- **政策**决定方向（钱从哪里来、往哪里去）
- **资金**决定力度（钱堆出来的行情才是真行情）
- **情绪**决定波动（同样的钱，情绪高涨时涨更多，情绪崩溃时跌更惨）

核心循环：
```
政策催化 → 资金预判 → 题材启动 → 龙头打出高度 → 情绪扩散 → 跟风入场 → 高位派发 → 情绪崩溃 → 退潮
```

## 二、资金层（Capital Allocation）

### 2.1 钱从哪里来

A股资金来源分类：

| 资金类型 | 行为特征 | 持股周期 | 代表 |
|---------|---------|---------|------|
| 机构（公募/私募） | 左侧布局，趋势强化 | 中线/长线 | 社保、险资 |
| 游资 | 题材催化，快进快出 | 短线 | 养家、赵老哥 |
| 外资（沪深港通） | 价值发现，跟随汇率 | 中线 | 北向资金 |
| 杠杆资金（两融） | 放大波动，顺势加仓 | 短线 | 融资客 |
| 散户 | 最后知后觉，跟风 | 不定 | 大多数人 |

### 2.2 资金的行为逻辑

- **赚钱效应是吸引资金的唯一原因**：有赚钱效应，存款、理财、房市资金都会搬家进来
- **亏钱效应是驱逐资金的核心因素**：亏钱效应一旦扩散，基金赎回、杠杆爆仓、散户销户接踵而来
- **存量博弈**：没有增量资金时，市场是零和游戏，钱从跟风者口袋流向先手者口袋
- **增量市 vs 减量市**：增量市（成交额持续放大）可以做趋势；减量市（成交额萎缩）只能做超跌反弹

### 2.3 关键指标

- **成交额**：两市成交额是市场温度计。<8000亿偏弱，>12000亿开始活跃，>15000亿牛市氛围
- **两融余额**：代表杠杆资金情绪，上升=多头加仓，下降=风险偏好收缩
- **北向资金**：外资风向标，持续流入=信心强，持续流出=信心弱
- **新开户数/基金发行量**：场外资金充裕度，决定行情能走多远

## 三、情绪层（Market Sentiment）

### 3.1 情绪周期的四个阶段

```
低位震荡 → 主升期 → 高位震荡 → 主杀期 →（循环）
```

| 阶段 | 情绪特征 | 赚钱效应 | 操作策略 |
|------|---------|---------|---------|
| **低位震荡** | 冰点，极度悲观 | 连板溢价低，炸板率高 | 轻仓试错，不追高 |
| **主升期** | 升温，风险偏好上升 | 连板溢价高，炸板率低 | 重仓龙头，积极做多 |
| **高位震荡** | 高潮，极度亢奋 | 龙头开始高位巨震 | 控仓，不追高位 |
| **主杀期** | 退潮，恐慌蔓延 | 核按钮、天地板频发 | 空仓或极轻仓 |

### 3.2 情绪传导链

核心原理：**亏钱效应从最强的那只股开始扩散**

1. 总龙头断板（最高板倒下）→ 情绪见顶信号
2. 跟风股跟跌 → 亏钱效应扩散
3. 连板股炸板率上升 → 短线资金开始收缩
4. 跌停家数增加 → 情绪加速恶化
5. 跌停家数达到峰值 → 冰点，可能反转

### 3.3 判断情绪的核心指标

| 指标 | 工具来源 | 说明 |
|------|---------|------|
| 涨停家数 | Tushare | >100只=偏热，<40只=偏冷 |
| 跌停家数 | Tushare | >30只=亏钱效应扩散 |
| 封板率 | VIP复盘网 | <60%=情绪差，>80%=情绪好 |
| 连板率 | VIP复盘网 | 连板/涨停 = 资金接力情绪 |
| 炸板率 | VIP复盘网 | 炸板/涨停 = 抛压情绪 |
| 涨跌比 | VIP复盘网 | 上涨/下跌家数比，>3:1多头强 |

### 3.4 养家心法核心

> "买入机会，卖出风险，只做对的交易，胜负交给概率"
> "买入倾向的推理，大多来自赚钱效应；卖出倾向的推理，大多来自亏钱效应"

理解：每次买入决策，本质上是在判断「现在市场的赚钱效应是否会持续」。赚钱效应在扩散就买入，赚钱效应在收缩就卖出。

## 四、题材层（Thematic Investment）

### 4.1 题材的本质

题材 = 故事 = 资金凝聚共识的方向

没有故事的股票只有基本面估值波动；有故事的股票有梦想溢价，可以涨过头。

### 4.2 好题材的四个维度（判断标准）

| 维度 | 说明 | 例子 |
|------|-----|------|
| **消息新** | 首次出现，认知差大 | 2026年AI应用爆发 |
| **大体量** | 涉及面广，影响产业链长 | 新能源车、半导体 |
| **高想象** | 远期空间大，短期无法证实也无法证伪 | 室温超导、AI |
| **强催化** | 政策/事件持续催化，逻辑不断强化 | 节假日预期、关税政策 |

### 4.3 题材的生命周期（五阶段）

```
预热期 → 发酵期 → 主升期 → 高潮期 → 退潮期
```

| 阶段 | 特征 | 操作 |
|------|-----|------|
| **预热期** | 消息首次出现，个股零星涨停 | 选股，观察 |
| **发酵期** | 板块开始联动，龙头确立 | 追进龙头 |
| **主升期** | 龙头连续涨停，打开空间 | 持有，或首阴低吸 |
| **高潮期** | 全板块鸡犬升天，补涨股乱飞 | 减仓，不追 |
| **退潮期** | 龙头断板，跟风股暴跌 | 清仓 |

### 4.4 题材轮动规律

资金在不同题材间移动的规律：

- **纵向轮动**（同一产业链）：电网设备→火电→算力协同→绿电/风电
- **横向轮动**（不同板块）：消息面新的板块卡位老的板块
- **高低切换**：主线退潮时，资金流向低位题材避险
- **情绪传导**：龙头股上涨→扩散到同概念小弟→再扩散到相关板块

### 4.5 判断主线是否延续

主线延续的条件（同时满足）：
1. 龙头股继续涨停，空间不断拓展
2. 跟风股跟随，板块内部轮动健康
3. 消息面持续有催化
4. 不出现新题材卡位

主线见顶信号：
1. 龙头断板（最重要）
2. 跟风股亏钱效应扩大
3. 新题材出现卡位
4. 情绪指标全面走弱

## 五、筹码层（Smart Money / 筹码运作）

### 5.1 主力运作的四个阶段

```
建仓（吸筹）→ 洗盘 → 拉升 → 出货
```

| 阶段 | 量能特征 | 价格特征 | 散户行为 |
|------|---------|---------|---------|
| **建仓** | 温和放量，量能堆积 | 区间震荡，底部抬高 | 割肉离场 |
| **洗盘** | 缩量下跌或横盘 | 回调但不破关键位 | 恐慌卖出 |
| **拉升** | 放量上涨，量价齐升 | 突破新高 | 犹豫买入 |
| **出货** | 高位放量震荡 | 滞涨或尖顶 | 高位接盘 |

### 5.2 关键筹码结构

- **倍量建仓**：主力连续放量进场，拿到底部筹码
- **地量洗盘**：缩量调整，清洗浮筹，为拉升扫清障碍
- **倍量启动**：放量涨停突破，拉开主升序幕
- **断板反包**：强势股调整后的二波启动，经典短线机会

### 5.3 筹码与情绪的共振

最暴力的机会：主力建仓完毕 + 情绪刚好转暖 + 题材消息催化
= 「主力控盘 + 资金共识 + 题材故事」三者共振

## 六、结构层（Price Patterns）

### 6.1 龙头股的结构特征

**总龙头特征**：
1. 启动最早，上涨最凶，调整最短
2. 能带动整个板块，甚至大盘共振
3. 期间有跟风小弟形成梯队
4. 断板=本轮行情顶部（总龙唯一性）

**连板梯队结构**：
- 4板+：总龙头候选，地位确立
- 3板：龙头确认，跟风资金最强
- 2板：资金接力确认，首板逻辑验证
- 1板：启动点，辨识度最重要

### 6.2 首板战法核心逻辑

首板是行情的起点，但也是最难识别的点。

**一进二（首板到二板）的核心逻辑**：
- 首板是资金试探，证明逻辑被认可
- 二板是接力确认，资金共识强化
- 关键：换手要健康（不是一字板），量能要够（放量但不过度）

**首板选择标准**：
- 涨停时间早（10:30前最佳）
- 封板强度高（一次性封死，硬板）
- 封单金额大（封单/流通市值比值高）
- 题材纯正（不是跟风上涨）

### 6.3 龙回头的结构

龙头股断板后，经过一段时间调整，再次上涨的行情。

**触发条件**：
- 龙头断板后回调幅度在15-25%之间（不破启动位）
- 回调时缩量（主力没跑）
- 人气仍在（讨论度高，资金记忆强）
- 市场情绪没有完全退潮

**买点**：回调至重要均线附近（5日/10日线），出现止跌K线

## 七、执行层（System & Psychology）

### 7.1 交易系统的核心原则

1. **只做龙头**：不是龙头不重仓，不做杂毛
2. **跟随为主**：不主观预测，只跟随市场选择
3. **止损优先**：不设止损线就不开仓
4. **情绪仓控**：根据情绪周期决定仓位大小
5. **空仓是高级操作**：退潮期不强行交易

### 7.2 仓位管理

| 情绪阶段 | 建议仓位 | 操作 |
|---------|---------|------|
| 冰点/低位震荡 | 0-20% | 轻仓试错 |
| 启动/回暖 | 50-70% | 积极做多 |
| 主升 | 80%+ | 重仓龙头 |
| 高潮/高位震荡 | 30-50% | 边打边退 |
| 退潮/主杀 | 0-10% | 空仓 |

### 7.3 每日复盘核心三问

1. **空间板是谁？生死如何？**（判断最高板情绪）
2. **主线板块是谁？延续还是退潮？**（判断资金聚集方向）
3. **亏钱效应在哪里扩散？**（判断风险）

### 7.4 常见认知偏差警示

| 偏差 | 表现 | 纠正 |
|------|-----|------|
| 后悔厌恶 | 卖飞的股又想追回来 | 只按系统操作，不后悔 |
| 损失厌恶 | 亏了死扛不卖 | 止损是呼吸，亏损不隔夜 |
| 确认偏误 | 只看支持自己持仓的信息 | 每天检查自己持仓的缺点 |
| 过度自信 | 重仓满仓梭哈 | 分仓，永不单吊 |
| 近期偏好 | 最近赚钱的方法就是好方法 | 系统化，不因短期结果改变体系 |

## 八、A股特殊规律（区别于其他市场）

1. **政策敏感**：重大政策可以瞬间改变行业预期
2. **T+1制度**：当日买入不能当日卖出，打板策略需要考虑次日兑现
3. **涨跌停板**：10%/20%限制创造独特的博弈环境（核按钮、翘板）
4. **散户占比高**：情绪波动更大，羊群效应更强
5. **消息不对称**：散户永远比机构后知道信息，所以要跟随而不是预判消息
6. **节假日效应**：节前避险情绪，节后开门红/开门黑规律

## 九、核心因果逻辑链（持续积累）

本框架的核心是理解「trigger → mechanism → outcome」的因果链，而非记忆招式。

以下是从学习内容中提炼的核心理由链，后续持续更新至 `logic_chains.json`：

---

**因果链01：龙头断板 → 行情见顶**

- **Trigger（触发）**：最高板（总龙头）断板
- **Mechanism（原理）**：总龙头是市场资金共识凝聚的最高点，它的倒下代表资金做多意愿的衰竭。跟风资金失去标杆，开始砸盘，亏钱效应扩散。
- **Outcome（结果）**：本轮行情情绪见顶，进入退潮期或高位震荡期。
- **When works**：在单题材主线行情中准确率高。
- **When fails**：主线切换时（旧龙断板但新龙启动），市场整体仍有机会。

---

**因果链02：情绪冰点 → 反弹酝酿**

- **Trigger（触发）**：跌停家数达到峰值（通常>30家），连板溢价率为负
- **Mechanism（原理）**：极度悲观情绪意味着抛压充分释放，割肉盘离场。聪明资金开始在低位试探性买入。
- **Outcome（结果）**：情绪从冰点回暖，强势股或新题材开始反弹。
- **When works**：市场整体非系统性崩溃时（如个股利空而非系统风险）。
- **When fails**：系统性风险（金融危机、疫情等）时，冰点之后还有冰点。

---

**因果链03：倍量涨停突破 → 主升浪启动**

- **Trigger（触发）**：个股以超过前期平均量能2倍以上的成交量涨停，突破关键压力位
- **Mechanism（原理）**：主力资金通过放量方式宣告进场，突破前期高点意味着上方无套牢盘压力，筹码结构干净。
- **Outcome（结果）**：连续涨停概率大幅提升，进入主升浪。
- **When works**：在主线题材内的个股，叠加情绪回暖期，准确率高。
- **When fails**：个股独立行情，无板块联动，可能是短庄自救。

---

**因果链04：缩量调整 + 板块联动 → 主力洗盘**

- **Trigger（触发）**：龙头股在连续涨停后，出现缩量调整（量能萎缩至峰值30%以下），同时板块内跟风股没有跟跌
- **Mechanism（原理）**：缩量说明主力筹码稳定，没有大规模出货。跟风股不跟跌说明板块主线地位仍在，资金看好后市。
- **Outcome（结果）**：主力洗盘结束后，大概率还有二波拉升。
- **When works**：调整幅度在15-25%，时间在3-5个交易日内。
- **When fails**：缩量跌破重要均线（如10日线），可能演变为真跌。

---

**因果链05：新题材卡位 → 旧主线退潮**

- **Trigger（触发）**：同一时期出现新题材，多只个股涨停，且板块效应强于现有主线
- **Mechanism（原理）**：市场存量资金有限，新题材的出现会吸引游资切换方向。资金从旧主线流出，流入新题材。
- **Outcome（结果）**：旧主线进入退潮期，新主线成为资金聚集方向。
- **When works**：新题材消息够新、想象空间大、能持续催化。
- **When fails**：新题材体量小，无法容纳大资金，只能是短暂过渡。

---

**因果链06：消息催化 → 资金抢筹 → 首板**

- **Trigger（触发）**：突发消息（政策、业绩、事件）刺激
- **Mechanism（原理）**：消息打破原有平衡，部分资金抢先手，在消息发酵的第一时间封板。首板是资金对消息认可度的第一次投票。
- **Outcome（结果）**：首板个股次日若能高开并快速二板，说明资金共识强。
- **When works**：消息足够重磅，市场整体氛围不差。
- **When fails**：消息不够重磅，或已有资金提前埋伏（见光死）。

---

**因果链07：赚钱效应扩散 → 场外资金入场**

- **Trigger（触发）**：龙头股连续涨停，短线选手普遍赚钱
- **Mechanism（原理）**：赚钱效应在圈子里传播，吸引新的散户和游资进场。增量资金进场进一步推高市场。
- **Outcome（结果）**：成交额放大，行情从存量博弈转为增量市。
- **When works**：龙头空间不断拓展，形成示范效应。
- **When fails**：行情涨得太快太急，监管降温或市场情绪透支。

---

**因果链08：期货/外盘联动 → A股次日开盘反映**

- **Trigger（触发）**：夜盘期货（原油、金属、农产品）或美股大跌
- **Mechanism（原理）**：A股有大量跟期货相关的周期股（能源、金属），外盘下跌影响其估值预期；美股是全球风险偏好的风向标。
- **Outcome（结果）**：A股次日大幅低开，盘中可能修复也可能继续下跌，取决于A股自身情绪。
- **When works**：A股处于弱势或高位时，外盘下跌的杀伤力更大。
- **When fails**：A股独立行情时（如政策强支撑），外盘影响有限。
