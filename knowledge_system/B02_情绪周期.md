# 情绪周期：短线交易的底层时钟

> 情绪周期是A股短线交易的底层时钟。一切交易决策都要先判断现在处于哪个阶段。

---

## 一、什么是情绪周期

情绪周期 = 市场参与者的群体心理在贪婪与恐惧之间往复循环的过程。

核心驱动：
- **赚钱效应** → 吸引资金入场 → 强化赚钱效应（正反馈）
- **亏钱效应** → 驱逐资金离场 → 强化亏钱效应（负反馈）

两个反馈循环交替，推动行情从启动到退潮。

---

## 二、四个基本阶段

```
低位震荡 → 主升期 → 高位震荡 → 主杀期 → 循环
```

### 阶段详解

#### 1. 低位震荡（冰点/筑底）

**情绪特征**：
- 市场极度悲观，成交量萎缩
- 涨停家数少（<40只），跌停家数多（>20只）
- 炸板率高（>40%），连板溢价低或为负
- 题材散乱，没有持续性主线

**本质**：亏钱效应的末端，割肉盘离场，抛压枯竭，但做多信心尚未恢复。

**操作策略**：
- 仓位：0-20%
- 选股：轻仓试错，不追高
- 机会：强势股超跌反弹，或新题材试探性买入
- 禁忌：重仓、抄底、YY（主观意淫）

---

#### 2. 主升期（赚钱效应扩散）

**情绪特征**：
- 市场情绪回暖，成交量温和放大
- 涨停家数上升（60-100只），跌停家数减少
- 封板率高（>75%），连板溢价明显
- 主线题材确立，龙头连续涨停
- 连板梯队结构健康（从1板到N板层层递进）

**本质**：赚钱效应开始扩散，资金风险偏好上升，龙头打出高度后吸引更多资金入场。

**操作策略**：
- 仓位：50-80%
- 选股：重仓主线龙头，不做杂毛
- 操作：追涨、持有、不轻易卖飞
- 关键点：判断主线是否延续，跟随而非预测

---

#### 3. 高位震荡（高潮/筑顶）

**情绪特征**：
- 市场极度亢奋，但上涨开始乏力
- 涨停家数维持高位（>80只），但炸板率上升
- 龙头高位巨震（振幅大、炸板频发）
- 跟风股开始掉队，亏钱效应开始出现
- 成交额达到峰值（可能是局部顶）

**本质**：多方力量达到顶峰，空方力量开始积蓄。龙头股滞涨说明资金接力意愿下降。

**操作策略**：
- 仓位：30-50%，边打边退
- 选股：只做总龙头，不碰跟风
- 操作：高位巨震时减仓，不追高位
- 关注：总龙头断板信号，一旦出现立即撤退

---

#### 4. 主杀期（退潮/崩溃）

**情绪特征**：
- 市场情绪急剧恶化
- 涨停家数骤降，跌停家数激增（>30只）
- 核按钮（天地板）频发，连板溢价转为负
- 主线龙头断板 → 本轮行情见顶
- 跌停家数持续增加，情绪全面崩溃

**本质**：亏钱效应扩散，资金恐慌踩踏，蒸发流动性。

**操作策略**：
- 仓位：0-10%
- 选股：空仓，不抄底
- 禁忌：接飞刀、买跟风、YY反弹
- 等待：跌停家数达到峰值 → 可能是冰点反转信号

## 三、细分的七个阶段（进阶版）

有的体系把周期分得更细：

```
冰点 → 混沌 → 启动 → 发酵 → 主升 → 高潮 → 退潮
```

| 阶段 | 特征 | 操作 |
|------|------|------|
| **冰点** | 跌停潮，极度悲观 | 等，不操作 |
| **混沌** | 情绪不稳定，方向不明 | 轻仓试错 |
| **启动** | 新题材出现，首板出现 | 跟进新题材 |
| **发酵** | 板块联动，龙头确认 | 追进龙头 |
| **主升** | 龙头加速，连续涨停 | 重仓持有 |
| **高潮** | 全民炒股，补涨乱飞 | 减仓 |
| **退潮** | 龙头断板，跌停扩散 | 清仓 |

## 四、判断情绪的核心指标

| 指标 | 冰点/低位 | 主升 | 高潮 | 主杀/退潮 |
|------|----------|------|------|----------|
| 涨停家数 | <40 | 60-100 | >80 | <40 |
| 跌停家数 | >20 | <10 | 增加 | >30 |
| 封板率 | <60% | >75% | 下降 | <50% |
| 炸板率 | >40% | <25% | 上升 | >50% |
| 连板溢价 | 负/低 | 高 | 高但下降 | 负 |
| 成交额 | 萎缩 | 放大 | 峰值 | 萎缩 |
| 最高板 | 2-3板 | 不断拓展 | 龙头高位震荡 | 断板 |

## 五、情绪传导链（原理）

理解情绪是如何传导的，才能真正把握周期：

```
① 总龙头打出高度（赚钱效应示范）
    ↓
② 资金模仿 → 跟风股上涨（赚钱效应扩散）
    ↓
③ 赚钱效应扩散 → 吸引增量资金
    ↓
④ 增量资金入场 → 进一步推高龙头
    ↓
⑤ 龙头滞涨（高位分歧）→ 亏钱效应萌芽
    ↓
⑥ 龙头断板 → 亏钱效应确认
    ↓
⑦ 跟风股恐慌 → 跌停扩散
    ↓
⑧ 亏钱效应扩散 → 资金收缩
    ↓
⑨ 跌停潮 → 冰点 → 循环
```

**核心传导节点**：
- 龙头断板 = 本轮情绪顶（最重要信号）
- 跟风股开始跌 = 亏钱效应从龙头扩散到跟风
- 跌停家数骤增 = 情绪崩溃加速
- 跌停家数达峰 = 可能是冰点，准备反转

## 六、高潮与退潮的区分（实战关键）

这是实战中最难判断的地方。

### 高潮期的特征

- 龙头还在涨，但越来越犹豫（高位震荡）
- 跟风股开始掉队（同一板块内部分化）
- 涨停家数多，但炸板率上升
- 成交量达到近期峰值

### 退潮期的特征

- 总龙头断板（最核心信号）
- 跌停家数超过涨停家数的30%
- 昨日涨停股今日溢价转负
- 核按钮（天地板）出现
- 连板股出现批量炸板

**记住**：高潮期还可以做补涨龙；退潮期任何股票都不安全。

| 情绪阶段 | 仓位 | 操作频率 |
|---------|------|---------|
| 冰点 | 0-20% | 低频 |
| 混沌/启动 | 30-50% | 试探 |
| 主升 | 50-80% | 积极 |
| 高潮 | 30-50% | 减仓 |
| 退潮 | 0-10% | 空仓 |

## 八、情绪周期的本质理解

> "只要人心不改变，人性不改变，情绪周期亦不会改变。"

人类的行为模式在贪婪与恐惧之间循环，这是情绪周期永恒存在的底层逻辑。无论市场如何进化，这个规律不会变。

**不同阶段的利润来源**：

- **低位/冰点**：利润来源于仓位（轻仓试错）
- **主升**：利润来源于持仓（重仓持有）
- **高位/高潮**：利润来源于选股（只做补涨龙）
- **退潮**：没有利润，只有风险

---

## 九、实战应用：三问判断法

每日前先问三个问题：

1. **空间板的生死？**
   - 总龙头是谁？连板了几板？
   - 今天有没有断板？
   - → 回答：现在是不是退潮期

2. **主线是否延续？**
   - 今日最强板块是谁？
   - 龙头有没有动摇？
   - → 回答：钱现在聚集在哪里

3. **亏钱效应在哪里扩散？**
   - 昨日涨停股今日溢价如何？
   - 今日跌停股有什么特征？
   - → 回答：现在的风险在哪里

## 十、龙空龙战法

情绪周期的高级应用：

**核心理念**：只在情绪主升期重仓龙头，其他时间空仓等待。

- **龙**：情绪主升期，重仓干龙头
- **空**：情绪退潮期，坚决空仓
- **龙**：下一轮周期启动，重新重仓

这个战法解决了两个问题：
1. 进攻（主升期重仓吃够）
2. 防守（退潮期空仓不亏）

**关键**：判断何时是"退潮"和"主升"的边界，这是最难的。
