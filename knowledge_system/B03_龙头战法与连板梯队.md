# 龙头战法与连板梯队

> 龙头战法是A股短线交易的核心。理解龙头，才能理解A股。

---

## 一、什么是龙头

### 定义

龙头 = 在某一段时间内，走出独立行情，连续涨停、快速翻倍、能带动板块升天、甚至带动大盘指数共振的个股。

### 三个层次的龙头

| 层次 | 定义 | 特征 |
|------|------|------|
| **总龙头** | 全市场最高板，主导一轮行情的核心 | 断板=本轮行情顶 |
| **板块龙头** | 某题材内涨幅最高、带动性最强的股 | 只影响本板块 |
| **补涨龙** | 主线龙头见顶后，资金挖掘的补涨标的 | 高度有限 |

---

## 二、龙头股的核心特征

### 2.1 涨幅特征

- **启动最早**：板块启动时第一个涨停
- **涨幅最大**：同一板块中涨幅最高的
- **调整最短**：上涨途中调整时间很短（通常1-3天）
- **最后倒下**：同一板块中最后一个开始跌的

### 2.2 结构特征

1. **有跟风小弟**：龙二、龙三形成梯队
   - 小弟跟风越积极，说明资金共识越强
   - 没有人气的板块即使涨停也不算真龙头

2. **板块联动**：能带动同板块个股上涨
   - 龙头涨，小弟跟涨 → 健康
   - 龙头涨，小弟不跟 → 孤掌难鸣

3. **有人气**：各平台讨论热度高
   - 同花顺/东方财富人气榜前列
   - 讨论度高=资金记忆强=二波容易

### 2.3 基本特征

- **流通市值适中**：太大拉不动，太小流动性差（通常10-100亿最佳）
- **价格低**：高价股散户不敢追，通常不超过20元
- **超跌位置**：在大盘下跌末端逆市涨停，说明主力强

---

## 三、龙头股的三个买点

### 买点一：首板（启动点）

**定义**：板块启动时，龙头股第一个涨停。

**逻辑**：龙头从首板开始确立地位，能第一时间参与的资金成本最低。

**条件**：
- 涨停时间早（10:30前）
- 封板强度高（一次性封死，硬板）
- 封单金额大（封单/流通市值比值高）
- 有板块联动（同一板块多只涨停）

**风险**：首板不确定性高，可能是假突破，需要次日验证。

---

### 买点二：二板（确认点）

> "二板定龙头，一板能看出来个毛。"

**定义**：首板后次日再次涨停。

**逻辑**：首板可能是偶然，二板是资金接力的确认。真正有题材支撑的个股才能二板。

**二板的分类**：

| 类型 | 特征 | 操作 |
|------|------|------|
| **缩量秒板** | 高开缩量快速封板 | 竞价买入确定性高 |
| **换手板** | 高开震荡后封板，成交量放大 | 等确认后买入 |
| **一字板** | 直接以涨停价开盘 | 开板后回封买入 |

**二板的核心逻辑**：二板是资金对首板逻辑的第二次投票。能够二板的股票，说明逻辑被更多资金认可。

---

### 买点三：分歧/首阴（回调点）

**定义**：龙头股在连续涨停后，第一次出现分歧（炸板/收阴线）。

**逻辑**：
- 短期涨太多，获利盘回吐 → 正常洗盘
- 如果洗盘后能继续涨停 = 主力控盘强 = 二波预期
- 首阴次日如果高开反包 = 弱转强 = 确定性买点

**买点**：
1. **首阴次日高开高走**：直接买入（弱转强确认）
2. **回调到5日/10日线**：缩量止跌时买入
3. **尾盘确认不跌**：盘中观察，尾盘买入

**核心条件**：
- 调整幅度不超过25%（跌太多主力可能出货）
- 调整时缩量（主力没跑）
- 人气仍在（讨论度没有明显下降）

## 四、连板梯队的结构

### 4.1 什么是连板梯队

市场中同时存在不同连板天数的股票，按高度排列形成梯队：

```
4板以上：总龙头候选
3板：    龙头确认
2板：    资金接力确认
1板：    启动点
```

### 4.2 梯队的实战意义

**梯队完整** = 做多热情高涨，行情健康
- 每一层都有股票 → 资金层层接力
- 越高位置的股 → 代表越强的资金共识

**梯队断裂** = 行情可能见顶
- 高位股先倒下 → 资金接力意愿下降
- 只有低位首板 → 没有资金愿意打高度

**怎么看梯队**：

| 现象 | 解读 |
|------|------|
| 4板、3板、2板、首板都有 | 梯队完整，行情健康 |
| 3板突然断层，只有2板和首板 | 高位接力意愿下降 |
| 最高板只有2板 | 市场情绪弱 |
| 最高板不断往上拓展 | 主升期 |
| 最高板高度开始下降 | 退潮信号 |

## 五、连板各阶段的核心逻辑

### 首板（1板）

**量能特征**：
- 平量首板：可能加速或一字板
- 放量首板：需看量比（1.5-2倍健康，过量=筹码不稳定）

**选股标准**：
1. 涨停时间早（10:30前）
2. 封板强度高（一次性封死）
3. 封单金额大
4. 题材纯正（不是跟风）

**本质**：资金对消息/题材的第一次试探性认可。

---

### 二板（2板）

**量能特征**：
- 缩量秒板：强势，筹码锁定
- 换手板：需要板块配合，量能要够

**核心逻辑**：
- 首板可能是试盘，二板才是真正确认
- 能二板的股票 = 题材逻辑被资金接力认可
- 二板后进入主升浪的概率大幅提升

---

### 三板（3板）

**量能特征**：
- 往往伴随爆量（换手增加）
- 如果缩量加速 = 主力高度控盘

**核心逻辑**：
- 三板是龙头地位确立的标志
- 三板以后 = 进入妖股行列
- 三板后操作策略以持有为主

---

### 四板及以上（4板+）

**量能特征**：
- 经典走势：高开 → 震荡 → 涨停（或炸板分歧）
- 四板/五板 = 往往出现分歧（市场分歧最大的地方）

**核心逻辑**：
- 四板是大部分个股的天花板（大部分股票止步4板）
- 能过四板的股票 = 真正有题材支撑的龙头
- 五板以上 = 妖股行列，赚钱效应最强

---

## 六、龙头见顶的信号

> "断板 = 本轮行情见顶"

### 判断龙头见顶的条件

| 信号 | 重要程度 | 说明 |
|------|---------|------|
| **总龙头断板** | ★★★ | 最核心信号，断板=行情顶 |
| **跟风股批量跌停** | ★★ | 亏钱效应扩散 |
| **新题材卡位** | ★★ | 资金切换方向 |
| **龙头高位巨震** | ★ | 主力分歧加大 |
| **成交量异常放大** | ★ | 可能是出货 |

### 龙头见顶后的三个阶段

1. **高位震荡**：龙头不再涨停，但也不跌停，震荡出货
2. **补跌**：龙头开始下跌，同板块跟风股恐慌
3. **退潮**：全市场情绪退潮，跌停潮出现

---

## 七、龙头战法的核心理念

> "高手买入龙头，超级高手卖出龙头。"

1. **只做龙头**：不是龙头不重仓，杂毛股利润薄、风险大
2. **有龙重仓，无龙空仓**：没有龙头机会时，坚决空仓等待
3. **跟随为主**：不YY买龙头，而是等市场告诉你谁是龙头
4. **止损果断**：龙头买错也要止损，不能死扛

---

## 八、弱转强的核心逻辑

**什么是弱转强**？

- 预期上涨，但次日竞价低开（弱）
- 低开后快速拉升，股价翻红/涨停（强）
- 这种"该弱不弱"的走势 = 主力控盘强 = 确定性好

**买点**：
1. 竞价低开幅度<2%
2. 开盘后快速翻红（5分钟内）
3. 拉升时量能配合

**本质**：弱转强是市场情绪的错误定价被纠正的瞬间，蕴含高赔率。

---

## 九、实战总结

**龙头战法操作清单**：

```
1. 判断大盘情绪（是否适合做多）
2. 找到主线板块（资金聚集在哪里）
3. 找到板块内最强股（谁涨得最多、谁封板最强）
4. 判断买点是首板/二板/分歧
5. 确认题材逻辑（为什么涨？有没有持续催化？）
6. 设止损线（5-7%）
7. 执行
```

**最忌讳的**：
- 不看大盘情绪，盲目追龙头
- 不看板块效应，买独立个股
- 不设止损，死扛亏损
- 在退潮期买龙头
